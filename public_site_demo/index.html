Slides
Slides are essentially "pages" of content. Only one slide shows on the screen at a time.
To create a slide, you must give the HTML element two classes:
  1. saber-slide to identify it as a slide
  2. saber-slide# to identify the index of the slide
    (remember: replace # with the actual slide number!)
An example slide with the index of 5 looks like:
<div class="saber-slide saber-slide5">I'm the fifth slide!<div>


Cool! How do I transition between slides?
Buttons
You transition from slide to slide using buttons.
To create a button you must give the HTML element two classes:
  1. saber-button to identify it as a button
  2. saber-buttonTo# to identify the slide number the button jumps to
    (remember: replace # with the actual slide number!)
An example button that jumps to slide 5 looks like:
<div class="saber-button saber-buttonTo5">I'm a button!<div>


Awesome. Can I tweak stuff?
Settings
The sabersettings.js file allows you to modify the easing, transition duration, and selectors used for slides and buttons.
It's all been kept very simple.


What about dependencies?
Dependencies
SaberSlides depends on
  • jQuery
  • jQuery Easing
  • jQuery scrollTo
All libraries are bundled with SaberSlides.
